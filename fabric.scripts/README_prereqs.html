Fabric Prerequisite Software

The following open source software packages are prerequisites for using the Fabric and must be downloaded before it is installed. Please note that IBM is not responsible for (and has no control over) these packages, and you will be required to accept each of their licenses. Drop the files into the Fabric's prereqs directory (fabric-root-directory/prereqs) and the Fabric runtime installer, described below, will place them into their correct locations for use. It is not necessary to unpack the zip files manually.

Note: the one exception to this is Mosquitto, which must be installed manually.

On Windows an unzip tool is also required. (For example 7zip). The environment variable ZIP must be set as the location of the 7za.exe file. The Windows .BAT file that uses ZIP can be adapted for other versions.

Mosquitto

Mosquitto MQTT Broker: mosquitto.org

The mosquitto commands to start and stop the broker should be used rather than fabadmin broker options.

For a quick start the mosquitto configuration should contain :

Default listener on port 1883
port 1883
A local listener for in-node messaging
listener 1884 127.0.0.1
Some suggested additional properties
max_inflight_messages 150
max_queued_messages 1000
retry_interval 60
Persistence setting
persistence_location {FABRIC_HOME}/brokers/{NODE_NAME}
For example.
persistence_location /opt/edgware-0.4.0/brokers/default/

Required Mosquitto download:
mosquitto.org/download - make sure to download and install the correct version for your platform.
