Using the Gaian Database with Edgware Fabric

The Gaian Database is a dynamic, distributed, federated database technology. It can be used as an alternative way to federate Edgware Fabric's distributed Registry. To use the Gaian Database the following software package must be downloaded before Edgware Fabric is installed. Drop the file into the prereqs directory (fabric-root-directory/prereqs). It is not necessary to unpack the file manually, the fabgaian script must be run to unpack and deploy the files.

Please note: you will be required to accept the Gaian Database license agreement to use the technology.

On Windows an unzip tool is also required. (For example 7zip). The environment variable ZIP must be set as the full path of the 7za.exe file. The Windows .BAT file that uses ZIP can be adapted for other versions.

The Gaian Database

Information on the Gaian Dynamic Distributed Federated Database, and a download, are available from: www.ibm.com/developerworks.

The software package can be obtained from: www14.software.ibm.com/webapp/iwm/web/preLogin.do?source=swg-gaian. Note that an IBM ID is needed to download this software.
