Introduction

This page provides a UI to test sending JSON messages to Edgware over HTTP.

Sample message: {"op":"register:platform-type","type":"Panther","desc":"Panther CLV from Iveco","correl":"2100"}

Multiple messages can be sent at the same time. For example: {"op":"register:platform-type","type":"Panther","desc":"Panther CLV from Iveco","correl":"2100"}, {"op":"register:platform","id":"BlackPanther","type":"Panther","desc":"An instance of Panther","correl":"2200"}

Messages can be delimited by any character other than "{" or "}"
